How to Create Illustrations with the Miro Board

Using the Miro Board to Create Illustrations

We provide a Miro board that contains all the basic elements that were used to create the explanatory illustrations on the Patterns. The Miro board provides a powerful way to visualize AR patterns by offering various pre-designed elements. These elements can be combined to create new and innovative AR examples and to collaborate on the requirements and implementation of the patterns into new applications and use-cases. This guide will walk you through how to use these elements to bring your AR concepts to life.

Build your own Scenarios

Scenario Elements

The Miro board contains a variety of elements that can be used to create AR scenarios. These elements include some Environments (indoor / outdoor), objects such as plants and tables, and perspectively correct anchors. Depending on the application, you may want to choose from different different representations of the same objects that fit the overall design. and interactive controls.
